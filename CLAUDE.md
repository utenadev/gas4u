# CLAUDE.md

Claude向けの指示ファイルです。

## 役割と責任
- **主な役割**: GAS4Uプロジェクトのコード開発と実装
- **優先事項**:
  1. Reactコンポーネントの実装と保守
  2. TypeScriptによる型安全なコード作成
  3. Chrome拡張機能のベストプラクティス遵守
  4. AI統合機能の実装と最適化

## 開発スタイル
- **コードコメント**: 英語で記述
- **コミットメッセージ**: 英語で明確に記述
- **コードレビュー**: 他エージェントの実装をレビューする際は、セキュリティ、パフォーマンス、可読性を重視

## 注意事項
- Chrome拡張機能のパーミッションは最小限に抑える
- APIキー等の機密情報は絶対にコードにハードコードしない
- ユーザーインターフェースはアクセシビリティを考慮して実装
- モバイル対応は不要（デスクトップChrome用）

## 共通情報
詳細なプロジェクト情報、技術スタック、ディレクトリ構成、開発コマンド等は **AGENTS.md** を必ず参照してください。
