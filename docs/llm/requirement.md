# GAS4U 要件定義

## 概要
Google Apps Script (GAS) を効率的に開発するための Chrome 拡張機能です。AI (Gemini) を活用し、コード生成、デバッグ、管理を支援します。

## コア機能

### 1. AI統合 (Gemini)
- **モデル**: Gemini Flash および Pro モデルをサポート。
- **機能**:
    - 自然言語プロンプトに基づくコード生成。
    - コード補完/支援。
    - デバッグ支援とエラー解説。
- **管理**:
    - Gemini API キーの安全な保存 (ローカル DB/Chrome Storage)。

### 2. GAS エディタ
- **インターフェース**: ES6+ JavaScript に適したモダンなコーディングインターフェース。
- **機能**: シンタックスハイライト、自動補完 (標準 + AI)。

### 3. ソース管理 & プロジェクト管理
- **Clasp 互換性**:
    - `clasp` 互換の Pull/Push 機能。
    - GAS プロジェクトの管理 (Project ID, Script ID)。
- **差分 & レビュー**:
    - ローカルコードと AI 生成コードまたはリモートコードを比較するビジュアル差分ツール。
    - AI による変更を適用する前のユーザー承認ワークフロー。

### 4. データ永続化
- **ローカルストレージ**: ブラウザストレージ (IndexedDB/Chrome Storage) を使用して以下を保存:
    - プロジェクトメタデータ。
    - API キー。
    - キャッシュされたソースコード。

## 技術的制約
- **プラットフォーム**: Chrome 拡張機能 (Manifest V3)。
- **依存関係**: GAS との対話には既存の `clasp` 実装 ( `./clasp` に配置) を使用する。
